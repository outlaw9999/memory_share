# 📚 Kit Examples

This directory contains examples for integrating and using `memory-share-kit` in your workflows.

## 📁 basic/
A minimal walkthrough of the core `init → learn → recall` flow.

## 📁 automation/
How to use Kit CLI in CI/CD pipelines or automated agent scripts.

## 📁 recovery/
Simulating and repairing memory drift using `kit doctor`.

---

## Running Examples

Most examples can be run directly if you have `memory-share-kit` installed:

```bash
pip install memory-share-kit
python examples/minimal_example.py
```
